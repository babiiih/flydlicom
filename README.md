# Flybird Game

Game web sederhana bergaya Flappy Bird dengan:

- sprite burung dari referensi,
- pipa biru dengan outline dan logo,
- background dengan watermark logo,
- kontrol klik/tap/spasi,
- tombol pause dan restart.

## Cara menjalankan

Buka `index.html` langsung di browser, atau jalankan server lokal:

```bash
python3 -m http.server 4173
```

Lalu buka:

```text
http://127.0.0.1:4173
```
